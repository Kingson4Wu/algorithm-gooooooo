
# 排序
1. [稳定]冒泡排序：
   + 时间：最坏-O(n^2)、平均-O(n^2)、最好-O(n)； 空间：O(1)
   + 相邻元素两两比较交换，双重循环；
   + 单次遍历没有交换的，提前结束全部遍历
   + length<=1 的数组不需要处理
2. [不稳定]快速排序：
   + 时间：最坏-O(n^2)、平均-O(nlog2n)、最好-O(nlog2n)； 空间：O(nlog2n)
3. [不稳定]堆排序：
   + 时间：最坏-O(nlog2n)、平均-O(nlog2n)、最好-O(nlog2n)； 空间：O(1)
